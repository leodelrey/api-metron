# api-metron
Flask Postgresql API

Frameworks :
- sqlalchemy
- marshmallow
- pytest
- tox

Linters :
- Mypy
- Flake8

# Create a virtual environment
The virtual environment here is flask folder
To run the API, you should create a new one and run :

      ```
      $ python -m flask
      $ pip install -r requirements.txt
      ```
